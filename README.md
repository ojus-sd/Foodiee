# 🍳 Recipe Finder

This is a simple web app I made where you can search for recipes easily. It uses the Spoonacular API to get recipe data.

---

## What it can do

- Search recipes by name (like “pasta”, “chicken”, etc.)
- Filter recipes by diet (vegetarian, keto, etc.) and cuisine
- Sort recipes (by time or alphabetically)
- Save your favourite recipes (they stay saved even after refresh using localStorage)
- Shows loading when fetching data and error message if something goes wrong

---

## Tech Used

- HTML  
- CSS (Flexbox & Grid)  
- JavaScript (Vanilla JS)  
- Spoonacular API
